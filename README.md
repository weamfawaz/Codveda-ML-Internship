# Task 3: K-Nearest Neighbors (KNN) Classifier

## Objective
Implement a KNN classifier to classify data points into categories.

## Dataset
- Iris Dataset
- Target variable: Species

## Steps Performed
- Data preprocessing and scaling
- Train/test split
- Training KNN classifier
- Model evaluation using accuracy, confusion matrix, precision, and recall
- Comparison of different K values

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Conclusion
The KNN classifier achieved high accuracy on the Iris dataset. Model performance varied with different values of K, demonstrating the importance of hyperparameter selection.
